# Папка с основными тетрадками

## simple_csv_analysis.ipynb

Проверка корректной вычитки данных из csv файлов.
Проверить, что вычитывается с нужной кодировкой, иначе будет кракозябра.

## ressearch_data

Базоая очистка данных от избыточных колонок, убраны nan, а также приведены даты и цены к нужным типам.
